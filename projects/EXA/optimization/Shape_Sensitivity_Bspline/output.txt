 Script to build shared object for input files

 Input: 1. name of working directory of source file and share object
        2. name of moris build directory under $MORISROOT
        3. name of input file (without cpp suffix)
        4. optional: keyword dbg to keep link to input file

 Output: .so file

 Example: 

 To create the shared objective file use:

 /home/soman/codes/moris_new/share/scripts/create_shared_object.sh . build myinput

 To run moris with this input file use:

 /home/soman/codes/moris_new/build/main/moris myinput.so
